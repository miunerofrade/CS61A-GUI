# CS61A GUI Windows 本地版

1. 将压缩包完整解压到一个有写入权限的目录，例如“文档”或课程作业目录。
2. 双击 `CS61A-GUI.exe`，浏览器会自动打开 `http://127.0.0.1:8761`。
3. 不要只把 EXE 单独移出文件夹；`_internal` 目录是程序的一部分。
4. 下载或导入的作业、缓存和备份会保存在本程序目录中。
5. 关闭命令行窗口即可停止本地服务。

本地版只监听 `127.0.0.1`，使用随软件附带的原生 Python 环境运行 OK 测试，适合
Scheme、SQL，以及需要线程、网络或课程图形界面的作业。

由于当前版本没有购买 Windows 代码签名证书，首次启动时 SmartScreen 可能显示
“Windows 已保护你的电脑”。请确认文件来自本仓库 Release，再选择“更多信息”与
“仍要运行”。Release 同时提供 SHA-256 校验文件。

遇到端口占用时，请先关闭已经运行的 CS61A GUI，再重新启动。
